## Recommended settings

<br>

<pre>
MIddle          = Default   
Right           = Default   
X1              = Default
X2              = Default
Left & Right    = Ctrl + W    (Close tab)
Right & Left    = None 
MIddle & Left   = Left + Alt  (History back) 
MIddle & Right  = Right + Alt (History forward)
